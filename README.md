# About
*SnakesSsSs* is a simple anonymous board

# Deploy
The project already contains docker-compose file, you just need to add .env to the root and do some config. Example:
```
POSTGRES_DB=db
POSTGRES_USER=admin
POSTGRES_PASSWORD=qwerty
HOST=12.13.20.2
```
